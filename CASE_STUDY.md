# Case Study: Rebranding a Cardiology Practice Website

> **Vireo Cardiology is a fictional brand** created for this portfolio case study.

*Roles: design + front-end. Stack: Next.js (App Router), React, Tailwind CSS v4, TypeScript.*

---

## Problem

_What was wrong with the original site / situation?_

- (e.g.) The original design used a fire-engine-red-on-near-black-navy palette that read as alarming rather than reassuring for a healthcare audience.
- (Context: this began as a proposal for a real practice that could not be reached, so it was repurposed under a fictional brand.)

## Goals

_What did the redesign set out to achieve?_

- Establish a calm, modern, premium identity ("Vireo Cardiology").
- Replace every reference to the original brand across code, copy, metadata, and structured data.
- Meet accessibility (WCAG AA+) and performance (Lighthouse ≥95) targets.

## Process

_How the work was approached._

- Defined a new palette (teal / coral / mist) and type system (Fraunces + Inter).
- Built a reusable logo component and design tokens in Tailwind's CSS-first theme.
- Redesigned hero, navigation, stats, cards, testimonials, and footer.
- Added a portfolio disclaimer, structured data, and SEO metadata.
